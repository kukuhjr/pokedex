# Pokedex
All Pokemons wiki app mobile first view. Sera x Avows test purpose.

Install dependencies:
```
  npm install
```

Run dev mode:
```
  npm run dev
```

### Tech
Pokedex uses a number of open source projects:

- ReactJS
- TailwindCSS
- HeadlessUI
- Axios
- React Query
- Vite
